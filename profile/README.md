# Welcome to Waffle Commons Ecosystem

**Waffle Commons** is an open-source organization dedicated to building modern, secure, and minimalist tools for PHP developers.

Our philosophy is rooted in the DevSecOps culture, where quality, security, and automation are at the heart of the development process.

# Our Projects
[waffle](https://github.com/waffle-commons/waffle): A modern, minimalist, and security-focused PHP micro-framework for building fast and reliable JSON APIs.
